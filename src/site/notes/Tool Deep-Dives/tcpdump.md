#### tcpdump
*tcpdump* is a lightweight, cross-platform^[called windump on Windows] packet capture tool. It can be fully automated to capture audit trail of all traffic

- `tcpdump -D`
	- Lists interfaces
- `tcpdump -XA`
	- X is for Hex
	- A is for ASCII
- `tcpdump -w`
	- w is to write the data to a file
