#### Linux Data Streams
- There are three data streams that can be used to move data around

| Name | accr | file designation |
| ---- | ---- | ---- |
| Standard In | stdin | 0 |
| Standard Out | stdout | 1 |
| Standard Error | stderr | 2 |
